# 記憶力向上ゲーム (Memory Match) - 説明書

## 概要
50代の方でも楽しみながら記憶力を鍛えられる、視認性とデザインにこだわった「神経衰弱ゲーム」です。
最新のアップデートにより、女子大生風の「大人可愛い」デザインと、やる気を引き出す「コンボ機能」が追加されました。

## 遊び方
1. **スタート**: ゲームを開くと、最初の3秒間だけ全てのカードが表向きになります（瞬間記憶タイム）。この間に配置を覚えましょう。
2. **めくる**: 裏返しになったカードをクリックしてめくります。
3. **ペアを揃える**: 2枚めくって同じ絵柄なら正解！そのまま表向きになります。
4. **コンボ**: 間違えずに連続で正解すると「コンボ」発生！スコアが倍増し、画面がキラキラと輝きます。
5. **クリア**: 全てのペアを揃えるとゲームクリア。ハイスコア更新を目指しましょう！

---

## 主な機能と変更点

### 1. 瞬間記憶モード
- ゲーム開始直後の3秒間、全カードが公開されます。
- 「運」ではなく「記憶力」でスタートダッシュを決めるチャンスです。

### 2. スコア・コンボシステム
- **基本スコア**: ペア正解でポイント獲得。
- **コンボボーナス**: 連続正解でスコアが「2倍、3倍…」とアップします。
- **ハイスコア保存**: 過去の最高得点が自動で保存されます。

### 3. 視覚演出の強化
- **JD Styleデザイン**: パステルピンクとラベンダーを基調とした、カフェ風の「大人可愛い」デザイン。
- **コンボ演出**:
    - **Gold Aura**: 2連続正解（コンボ確定）すると、揃えたペアが一斉に金色に輝き始めます。
    - **演出のメリハリ**: コンボが途切れると光も消えるため、現在の調子が直感的にわかります。
- **カードアニメーション**:
    - **回転**: 目が回らないよう、シンプルな「パタン」という動きに調整しました。
    - **絵柄の表示**: カードが真横（90度）を向いた瞬間に絵柄が変わるため、まるで本当に裏返しているような自然な見た目です。

### 4. 安心の設計
- **1ファイル完結**: 面倒なインストール不要。`index.html` を開くだけですぐに遊べます。
- **エラー対応**: 万が一のエラー時も、画面にわかりやすくメッセージが表示されます。
