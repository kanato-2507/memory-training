# 記憶力トレーニング v4 🧠✨

カードをめくって同じ絵柄を当てる、  
シンプルだけど奥が深い「記憶力トレーニングゲーム」です。

v4では、スコア・コンボ・演出を強化し、  
**遊びごたえがグッと増した進化版**になっています。

---

## 🎮 遊び方
1. カードを2枚ずつめくります  
2. 同じ絵柄なら成功  
3. すべて揃えたらクリアです  

できるだけ少ない手数＆高スコアを目指しましょう。

---

## 🚀 v4の特徴

### 🔥 コンボ＆スコアシステム
- **コンボ機能**  
  間違えずに連続で正解すると「コンボ」発生。  
  スコアが倍増し、画面がキラキラと輝きます。
- **コンボボーナス**  
  連続正解でスコアが **2倍、3倍…** とアップ。  
  集中力が続くほど、気持ちよさも加速します。

### 🎨 視覚演出の強化
- **JD Styleデザイン**  
  パステルピンクとラベンダーを基調にした、  
  カフェ風の「大人かわいい」デザイン。
- **Gold Aura演出**  
  2連続正解でコンボ確定時、  
  揃えたカードが金色のオーラに包まれて輝きます。

### 🧩 その他
- スコア表示あり
- ハイスコアを保存して、何度でも挑戦可能
- インストール不要（ブラウザですぐ遊べる）
- スマホ・PCどちらでもOK
